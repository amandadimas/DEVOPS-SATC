**DevOps:**
DevOps é uma cultura e conjunto de práticas que integram desenvolvimento e operações para entregar software de forma mais rápida, automatizada e confiável.
Envolve CI/CD, automação, infraestrutura como código e monitoramento contínuo.

**SRE (Site Reliability Engineering):**
SRE é uma abordagem que aplica engenharia de software à operação de sistemas, com foco em disponibilidade, performance e confiabilidade.
Trabalha com métricas como SLO, SLA e controle de error budget para garantir estabilidade em produção.


# 🧮 Calculadora Básica – React

A Calculadora Básica é uma aplicação desenvolvida em **React**, com o objetivo de realizar as quatro operações matemáticas principais: adição, subtração, multiplicação e divisão.

A interface é composta por um visor e botões numéricos e de operações. A lógica dos cálculos é controlada por meio de estados (useState), permitindo atualizar o valor exibido na tela conforme o usuário interage com os botões.

Este projeto demonstra conceitos fundamentais do React, como componentes, estados e manipulação de eventos, aplicados em uma aplicação simples e funcional.
