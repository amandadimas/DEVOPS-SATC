**DevOps:**
DevOps é uma cultura e conjunto de práticas que integram desenvolvimento e operações para entregar software de forma mais rápida, automatizada e confiável.
Envolve CI/CD, automação, infraestrutura como código e monitoramento contínuo.

**SRE (Site Reliability Engineering):**
SRE é uma abordagem que aplica engenharia de software à operação de sistemas, com foco em disponibilidade, performance e confiabilidade.
Trabalha com métricas como SLO, SLA e controle de error budget para garantir estabilidade em produção.
